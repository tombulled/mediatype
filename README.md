# mediatype
Media Type parsing and creation

## Usage
```python
>>> import mediatype
>>>
>>> mime = mediatype('application/manifest+json')
>>>
>>> mime
MediaType(type='application', subtype='manifest', suffix='json', parameters=None)
>>>
```
